# Hermes Integration Plan

Status: Draft  
Owner: Bryant Bishop  

## Purpose

This document explains how Hermes / Orion integrates with AVOS.

AVOS defines the operating framework.  
Hermes / Orion implements parts of that framework as an AI assistant.

## Relationship

AVOS is the source of truth for:

- Operating principles
- Scheduling rules
- Calendar ownership
- Assistant behavior standards
- Architecture decisions
- Implementation guidance

Hermes / Orion is responsible for executing approved workflows within those rules.
