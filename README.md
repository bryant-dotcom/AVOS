# AVOS

AVOS stands for AVseven Operating System.

AVOS is a documentation-first operating framework for organizing business systems, AI agents, automation, scheduling, identity, and implementation standards.

This repository is the source of truth for the AVOS framework.

## Purpose

AVOS separates business operating principles from the specific tools used to implement them.

Tools may change.  
Architecture should endure.

## Current Focus

- Scheduling system of record
- Calendar ownership rules
- Hermes / Orion integration
- AI assistant operating standards
- Documentation structure
- Implementation playbooks
