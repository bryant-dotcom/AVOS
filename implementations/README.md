# Implementations

Status: Draft  
Owner: Bryant Bishop  

## Purpose

This folder documents real-world implementations of AVOS.

AVOS is the framework.  
Implementations are the specific systems, agents, tools, and workflows that apply the framework.

## Current Implementations

### Hermes / Orion

Hermes / Orion is the first AVOS agent implementation.

It currently uses:

- Telegram for interaction
- GitHub for code and version control
- Hostinger for hosting and services
- Google Calendar for scheduling

## Rule

Implementation details should not override AVOS operating rules.

When an implementation creates a new pattern or decision, that decision should be documented in the appropriate AVOS file.
