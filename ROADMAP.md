# AVOS Roadmap

Status: Draft  
Owner: Bryant Bishop  

## Purpose

This roadmap tracks the phased development of AVOS from Bryant’s internal operating system into a reusable framework.

## Phase 1: Foundation

Goal: Establish the core AVOS documentation structure.

Status: In Progress

Includes:

- README
- Vision
- Principles
- ADRs
- Scheduling
- Calendar inventory
- AI assistant rules
- Hermes / Orion implementation notes

## Phase 2: Scheduling System

Goal: Make Google Calendar the clear scheduling source of truth.

Status: In Progress

Includes:

- Clean calendar inventory
- Approved calendar ownership
- AVseven calendar sync confirmed
- Hermes calendar rules documented
- Booking page rules defined

## Phase 3: Hermes / Orion Integration

Goal: Connect Hermes to the approved scheduling ecosystem.

Status: Next

Includes:

- Read-only calendar access
- Telegram calendar commands
- Controlled event creation
- Event update and cancellation safeguards
- Logging of calendar actions

## Phase 4: Automation Layer

Goal: Define how n8n and other automation tools interact with AVOS.

Status: Future

Includes:

- Webhooks
- Scheduled jobs
- Nightly capture
- Notifications
- Workflow documentation

## Phase 5: Business Owner Framework

Goal: Turn AVOS into a reusable framework for other CEOs, founders, and business owners.

Status: Future

Includes:

- Templates
- Setup guides
- Operating playbooks
- Implementation checklist
- Client-facing explanation

## Guiding Principle

Build first as a working reference implementation.  
Then convert the working system into a reusable framework.
